FORUM 1.9.3 documentation¶

Welcome!

What is FORUM: a test tool for telecommunication branch written in Python programming language developed by noriba GmbH company. Forum aims on testing of devices connected to PC via VISA interface, such TCPIP, GPIB, USB, ASRL, TELNET, etc. and supports SOAP communication. Application consists of built-in Python interpreter, giving the possibility to use huge amount of Python libraries in script itself. Maintenance of test results with Reporter offers clean overview and coordinates test planning.
